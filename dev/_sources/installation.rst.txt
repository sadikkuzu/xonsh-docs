
Installation
============

Prerequisites
-------------
* python V3.6 or later.
* for interactive use, an ansi (vt100-compatible) terminal application.  On Windows 10, the 
  separately-installable `Windows Terminal app <https://github.com/microsoft/terminal/releases>`_ 
  is recommended.
  
Installation
------------
